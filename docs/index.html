OMNIA

Post-hoc structural measurement and gating core.

OMNIA does not judge meaning. It does not replace reasoning. It does not train a model. It measures whether an output that looks acceptable on the surface remains structurally admissible under bounded variation.

surface-readable output != always structurally admissible

In this bounded 8-case LLM-like readable set, OMNIA assigned only 3/8 cases to GO and flagged 5/8 as structurally non-GO.

Why this matters

A readable answer can still be structurally fragile.

OMNIA is a bounded gate for detecting that divergence after inference, without requiring semantic interpretation as part of the core.

Reproduce in one minute

pip install -e . -U --no-cache-dir
python examples/run_profiles_jsonl.py examples/llm_surface_cases.jsonl -o examples/llm_surface_results.jsonl
python examples/analyze_llm_surface_results.py

Expected result: 8 surface-readable cases, 3 GO, 5 non-GO.

What this does not claim

  • Not a broad benchmark claim
  • Not a production-readiness claim
  • Not a semantic correctness claim
  • Not a safety guarantee

This page shows one bounded public case only.
