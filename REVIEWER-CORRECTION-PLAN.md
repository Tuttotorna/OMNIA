---

## Reviewer correction

The strongest technical core is not the claim that OMNIA is a new mathematical foundation.

The strongest technical core is this operational condition:

A projection is valid only for the decisions it preserves.

Let:

`Omega` = accessible field / system / generative structure

`pi: Omega -> P` = projection, metric, benchmark, score, KPI, aggregate, model, representation, or output

`D: Omega -> A` = decision, inference, classification, ranking, judgement, or action

The projection `pi` is valid for `D` only if:

`D = d composed with pi`

for some function:

`d: P -> A`

Equivalently:

`pi(omega1) = pi(omega2) implies D(omega1) = D(omega2)`

Failure condition:

`pi(omega1) = pi(omega2)` and `D(omega1) != D(omega2)`

If this occurs, then `pi` cannot ground `D`.
