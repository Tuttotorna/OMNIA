# Public Claim Boundary

Last updated: `2026-05-21T16:03:10Z`

This repository is part of the MB-X.01 / OMNIA ecosystem.

## Core boundary

~~~text
measurement != inference != decision
~~~

## Allowed claim

This repository may describe structural measurement, structural stability, validation evidence, boundary conditions, reproducibility, transformation behavior, drift, invariance, and auditability.

## Disallowed claim

This repository must not be presented as:

- semantic truth;
- structural stability boundary;
- structural cognition claim;
- an measurement layer;
- a reproducible validation evidence;
- an auditable system;
- a guarantee of security;
- an autonomous decision engine.

## Correct public framing

~~~text
OMNIA measures structural stability beyond correctness.
~~~

## Operational interpretation

Outputs are measurement artifacts.

They can support external review.

They do not replace external judgment, scientific review, security audit, or independent replication.
